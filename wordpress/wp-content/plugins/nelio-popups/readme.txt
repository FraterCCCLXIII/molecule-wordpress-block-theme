== Description ==

= THE EASIEST WORDPRESS POPUP BUILDER FOR THE BLOCK EDITOR =

Molecule Popups is a fork of Nelio Popups with all premium features enabled. Create beautiful popups, modals, and content overlays using the WordPress block editor.

= Features (all included) =

* Create and manage unlimited popups
* Fullscreen popups for newsletters and marketing
* Intelligent conditions: pages, cookie, referrer, device, browser, OS, language
* Adblock detection - show popups when adblock is detected
* New vs returning visitor targeting
* Date and time scheduling
* Window width conditions
* Query parameter targeting
* WooCommerce cart conditions (when WooCommerce is active)
* Age gate integration (Molecule/Shadcn theme)
* Frequency controls: first-time only, cooldown periods
* Rounded corners and custom styling
* Exit intent, scroll, time delay, click, and hover triggers
* Sound effects
* Block editor integration - use any block in your popups

= Fork Notes =

Molecule Popups is a community fork. Premium upsell and Nelio branding have been removed. All features are free.

== Installation ==

1. Upload the plugin folder to `/wp-content/plugins/`
2. Activate through the 'Plugins' menu in WordPress
3. Go to Nelio Popups (or Popups) in the admin menu to create your first popup

== Frequently Asked Questions ==

= Can I use Molecule Popups without the block editor? =

No. The plugin relies on the block editor (Gutenberg) that comes with modern WordPress versions.

= Is this a paid plugin? =

No. Molecule Popups is fully free. All features from the original premium version are included.
